# VapourSynth-DFTTEST2_HIPRTC

This package contains the HIPRTC implementation of the [vs-dfttest2](https://github.com/AmusementClub/vs-dfttest2) implemetation.

## Installation

```bash
pip install vapoursynth-dfttest2-hiprtc
```

## Building from source

```powershell
uv build --package vapoursynth-dfttest2-hiprtc
```
