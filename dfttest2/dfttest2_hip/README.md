# VapourSynth-DFTTEST2_HIP

This package contains the HIP implementation of the [vs-dfttest2](https://github.com/AmusementClub/vs-dfttest2) implemetation.

## Installation

```bash
pip install vapoursynth-dfttest2-hip
```

## Building from source

```powershell
uv build --package vapoursynth-dfttest2-hip
```
