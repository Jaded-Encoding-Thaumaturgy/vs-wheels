# VapourSynth-DFTTEST2_GCC

This package contains the GCC implementation of the [vs-dfttest2](https://github.com/AmusementClub/vs-dfttest2) implemetation.

## Installation

```bash
pip install vapoursynth-dfttest2_gcc
```

## Building from source

```powershell
uv build --package vapoursynth-dfttest2_gcc
```
